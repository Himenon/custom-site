## JavaScript / CSSのローカルスコープ参照

この記事はJavaScript、CSSがローカルスコープの参照を持っています。

```yaml
title: ローカルスコープ機能
description: CustomSiteが提供するローカルスコープ機能の紹介
js:
  - ./assets/local.js
css:
  - ./assets/local.css
```

このように記述することで、現在のURLから相対的な位置にあるディレクトリを参照し、JS/CSSの影響範囲を局所的にしています。
